本文件定义项目的化学安全、文件系统安全和秘密信息保护边界。

# 安全说明

## 化学安全边界

“化实”支持正常教学内容：实验目的、原理、常见仪器、教学性步骤、预期现象、化学方程式、PPE、通风、废液处理和常见失败排查。

以下请求不提供可直接执行的配方、比例、装置参数、温压条件、纯化浓缩或规模化细节：

- 爆炸物、起爆剂和高能材料制备。
- 大规模有毒气体制备或释放。
- 高毒化学品的提纯、浓缩或隐蔽获取。
- 非法药物、受控物质或有害物质合成。
- 规避实验室安全、监管、检测或追踪。
- 明显伤害本人或他人的操作。
- 未成年人在无教师监督下实施的高风险实验。

拒绝响应会简要说明风险，并提供非操作性的原理、虚拟实验、低风险演示或教师监督建议。规则系统是学习版防护，不是专业风险评估。

## 文件系统安全

- 写入仅限 `workspace/outputs/`。
- 读取仅限 `workspace/inputs/`。
- 解析结果仅进入 `workspace/parsed/`。
- 所有路径均在 `resolve()` 后验证父目录。
- 拒绝绝对路径、`..`、目录分隔符、隐藏文件、符号链接和系统保留名。
- 写文件仅支持 `md/txt/json`，JSON 先校验。
- 读文件检查扩展名、存在性、非空和大小上限。
- MinerU ZIP 解压逐项检查目标路径，防止 ZIP Slip。
- 大型文档只向模型提供摘要和截断片段。
- Web 上传采用分块读取并执行相同大小限制；重名文件使用随机后缀，不覆盖原文件。
- 上传接口拒绝隐藏文件、控制字符、不支持扩展名和路径穿越文件名。

## API Key 与日志安全

- 项目只提供空值 `.env.example`。
- 密钥字段使用 Pydantic `SecretStr`。
- 代码、测试、文档和验证报告不包含真实密钥。
- LangSmith 仅在显式启用且 Key 存在时开启。
- 默认日志不打印请求头、Token、完整文件或模型上下文。
- `/api/health` 只返回能力布尔值，不返回环境变量内容。
- Web 流式异常在非调试模式下不包含内部异常详情或堆栈。
- 调试模式只输出必要错误信息；仍不应打印秘密字段。
- `.gitignore` 排除 `.env`、缓存、用户文件、解析产物和日志。

## 责任边界

本项目不能替代教师、实验室负责人、机构安全规程、安全数据表（SDS）、医疗建议或应急处置。出现暴露、火灾、泄漏或人员不适时，应立即遵循所在机构的应急流程并联系合格人员。
